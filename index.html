PROJETOS

Projetos que tive a oportunidade de desenvolver e colocar em prático os conhecimentos obtidos.

De Bairro

Desenvolvi, em colaboração com meu grupo, um aplicativo para o gerenciamento de pequenos comércios, com funcionalidades como controle de estoque, cadastro de clientes e gestão financeira. Utilizamos Android Studio e Java, aplicando metodologias ágeis como Scrum e Kanban para garantir eficiência no desenvolvimento.

Competências: Java · SQL · Desenvolvimento Android · Trabalho em equipe

FlexTrek

Desenvolvi um sistema para uma loja de artigos esportivos, visando otimizar o cadastro, pesquisa e gestão de produtos e fornecedores. O projeto foi realizado em Java, com banco de dados MySQL, utilizando o modelo MVC e programação orientada a objetos (POO). A prototipagem foi feita no Figma para um design mais eficiente.

Competências: Java · SQL · POO · Prototipagem

PlantFy

Um sistema inovador que conecta a tecnologia à natureza, permitindo monitorar a umidade das plantas e otimizar o cuidado delas com inteligência artificial. O sistema também inclui uma API de libras e segue as 10 Heurísticas de Nielsen para uma experiência acessível e intuitiva.

Competências: HTML5 · CSS · JavaScript · SQL · PHP · C++
